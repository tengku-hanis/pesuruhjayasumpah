Emojis podem ser ativados em um projeto Hugo de diversar formas.

A função emojify pode ser chamada diretamente nos templates ou com Inline Shortcodes.

Para ativar os emojis globalmente, aplique enableEmoji com o valor true na configuração do seu site e então você poderá adicionar códigos de emoji diretamente nos arquivos de conteúdo. Por exemplo:

O Emoji cheat sheet é uma referência útil para identificar códigos de emojis.

Nota: Os passos anteriores ativam os caractéres e sequências de emoji do Padrão Unicode no Hugo, porém a renderização desses glifos depende do navegador e plataforma utilizada. Para escolher o estilo dos emojis você pode tanto usar um font de emoji de terceiros ou uma lista de fontes. Por exemplo:

.emoji {
font-family: Apple Color Emoji, Segoe UI Emoji, NotoColorEmoji, Segoe UI Symbol, Android Emoji, EmojiSymbols;
}
